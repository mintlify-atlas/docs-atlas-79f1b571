      Quick Start
    </h2>
    <p className="text-base dark:text-gray-400 text-gray-600 mb-8">
      Get Film Fanatic running locally in just a few steps
    </p>

    <Steps>
      <Step title="Clone and Install">
        Clone the repository and install dependencies with your preferred package manager.

        <CodeGroup>
        ```bash npm
        git clone https://github.com/Swastikdan/Film-Fanatic.git
        cd film-fanatic
        npm install
        ```

        ```bash pnpm
        git clone https://github.com/Swastikdan/Film-Fanatic.git
        cd film-fanatic
        pnpm install
        ```

        ```bash yarn
        git clone https://github.com/Swastikdan/Film-Fanatic.git
        cd film-fanatic
        yarn install
        ```
        </CodeGroup>
      </Step>

      <Step title="Configure Environment Variables">
        Create a `.env.local` file with your API keys and service credentials.

        ```env .env.local
        # TMDB API
        VITE_TMDB_ACCESS_TOKEN=your_tmdb_token_here
        VITE_TMDB_API_URL=https://api.themoviedb.org/3

        # Clerk Authentication
        VITE_CLERK_PUBLISHABLE_KEY=your_clerk_key_here
        CLERK_SECRET_KEY=your_clerk_secret_here

        # Convex Backend
        CONVEX_DEPLOY_KEY=your_convex_key_here
        VITE_CONVEX_URL=your_convex_url_here
        ```

        <Note>
          You'll need accounts with [TMDB](https://www.themoviedb.org/settings/api), 
          [Clerk](https://clerk.com), and [Convex](https://www.convex.dev) to get your API keys.
        </Note>
      </Step>

      <Step title="Start Development Server">
        Run both the Vite dev server and Convex backend simultaneously.

        ```bash
        pnpm dev
        ```

        This runs both `vite dev` and `convex dev` in parallel. Your app will be available at `http://localhost:3000`.
      </Step>

      <Step title="Explore the Platform">
        Open your browser and start exploring! Search for movies and TV shows, create your watchlist, 
        and track your viewing progress.

        <Tip>
          Guest users can use the platform with local storage. Sign in with Clerk to enable 
          cloud synchronization and access your watchlist from any device.
        </Tip>
      </Step>
    </Steps>
